**G1 News Scrapper** 📰🐍 **Status:** Concluído/Ferramenta de Automação

Um script robusto desenvolvido em **Python** para extração automatizada de manchetes e conteúdos do portal de notícias G1. Este projeto foca em eficiência de coleta e estruturação de dados não estruturados da web.

⚙️ **Funcionalidades**
- **Extração Seletiva:** Coleta títulos, subtítulos e links das principais notícias;
- **Filtragem por Categoria:** Se houver, por exemplo: Tecnologia, Economia, etc;
- **Exportação de Dados:** Salva os resultados em formato CSV para análise posterior;
- **Tratamento de Erros:** Lógica para lidar com variações no DOM e evitar interrupções durante a execução.

🛠 **Tecnologias**
- **Linguagem:** Python 3.x
- **Bibliotecas Principais:** BeautifulSoup4 (parsing HTML) e Requests (requisições HTTP).

🚀 **Como Usar**
1. Instale as dependências no terminal: pip install beautifusoup4 requests;
2. Execute o script: python main.py
